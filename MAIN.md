# ===PasteME - originex.tech - CLI Pastebin===

## What is it?

PasteME is a pastebin service designed for users who need to quickly copy logs or other system files using a single command.

## General operating principles (and limitations)

Since this project is currently neither popular nor well-funded, there are a few limitations:

- I am currently using the free plan on Upstash (max 256 MB and 500k commands/requests per month), but I'll move the project to my VPS once it gains popularity
- The server is hosted on my VPS
- The max size of a single paste is 2 MB
- The max expiration time for the paste is 24 hours (this can be reduced)
- I'm currently considering introducing one-time pastes or password-protected pastes, but for now, there won't be any

## Instructions

Since I currently do not have access to systems like Linux or macOS, the commands I prepare for them will be generated by AI.

### Windows (PowerShell because why not)

1. The basic command:

`(Get-Content file | Invoke-WebRequest -Uri "http://originex.tech" -Method Post).Content`

2. It can be extended by the expiration time:

`(Get-Content file | Invoke-WebRequest -Uri "http://originex.tech" -Method Post -Headers @{"X-TTL"="300"}).Content`

**Remember to replace the "file" field with your own file, and note that the max expiration time for the paste is 24 hours (86400 s).**

### Windows (CMD)

1. The basic command:

`curl --data-binary @file http://originex.tech`

2. It can be extended by the expiration time:

`curl -H "X-TTL: 300" --data-binary @file http://originex.tech`

**Remember to replace the "file" field with your own file, and note that the max expiration time for the paste is 24 hours (86400 s).**

### Linux (AI generated)

`cat file | curl --data-binary @- http://originex.tech`

2. It can be extended by the expiration time:

`cat file | curl -H "X-TTL: 300" --data-binary @- http://originex.tech`

**Remember to replace the "file" field with your own file, and note that the maximum expiration time for the paste is 24 hours (86400 s).**
